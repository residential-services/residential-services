Equa ergänzt Ihr ERP-System mit mieterspezifischen Functionen

  • Equa
  • Produkteigenschaften
  • Anwendung für Liegenschaftsverwalter

01.

Entsendung von Event-Ankündigungen

Event-Ankündigungen können den Mietern einer Gruppe von Gebäuden, einer bestimmten Gebäude oder einer einzelnen Wohneinheit gesendet werden. Sie beziehen sich auf zeitbegrenzte Events wie zum Beispiel eine Reparatur. Mieter können dann relevante Events mittels des Equa-Apps oder einer Kalenderanwendung zur Kenntnis nehmen, da Mieterkonten mit gängigen Kalendardiensten synchronisiert werden können.

02.

Entsendung von Dokumenten

Dokumente können den Mietern einer Gruppe von Gebäuden, einer bestimmten Gebäude oder einer einzelnen Wohnung gesendet werden. Ein Dokument kann zum Beispiel Hausregeln oder einen Mietvertrag enthalten.

03.

Feedback von Mietern

Mieter können Ihnen aus verschiedenen Gründen Feedback schicken, beispielsweise um einen mangelhaften Gerät anzumelden.

04.

Mietergruppen

Sie können Mietergruppen erstellen und dann mit ihnen effizient kommunizieren. Equa importiert die in Ihrem ERP-System enthaltenen Gruppierungdaten automatisch. Gruppen können auch innerhalb von Equa erstellt werden.

05.

Gemeiname Waschräume

Sie können Equa darauf hinweisen, ob bestimmte Gebäuden mit Waschräumen ausgestattet sind, die für Mieter zur Verfügung stehen. Equa importiert die in Ihrem ERP-System enthaltenen Waschraumdaten automatisch. Diese Daten können auch innerhalb von Equa erstellt werden. Equa erzeugt Waschraumreservierungdaten automatisch für jeden einzelnen Mieter. Reservierungen können zu jeder Zeit von den Mietern auch geändert werden. Mieter können dann ihre Reservierungen mittels des Equa-Apps oder einer Kalenderanwendung zur Kenntnis nehmen, da Mieterkonten mit gängigen Kalendardiensten synchronisiert werden können.

06.

Aktuelle Kontaktdaten

Sie können Equa darauf hinweisen, welche Personen für Mieter zur Verfügung stehen. Equa importiert die in Ihrem ERP-System enthaltenen Kontaktdaten automatisch. Diese Daten können auch innerhalb von Equa erstellt werden.

Datenschutz

Equa ist 100% konform GDPR gegenüber.

Installation

Equa ergänzt Ihr bestehendes ERP-System. Zur Zeit unterstützen wir AbaImmo und Quorum. Für AbaImmo-Benutzer stellen wir die Equa Connector Software zur Verfügung, die zusätzlich für Sie installiert und konfiguriert wird. The Equa Connector ist für macOS und Microsoft Windows erhältlich.
